class intervene.bcause.ordermcmc.scoring.BGe(data, nu=None, alpha_mu=1.0, alpha_omega=None, prior_matrix=None)¶

Bases: intervene.bcause.ordermcmc.scoring.continous_score_functions.ContinuousData

Class to compute the BGe function for Bayesian Networks.

Create a BGe scoring object.

Parameters

  • data (numpy.ndarray) – The data to use for scoring

  • nu – the mean vector for the normal part of the normal-Wishart prior. If not given (=None), then the sample mean is used.

  • alpha_omega (int) – The degrees of freedom for the Wishart part of the normal-Wishart prior. Must exceed p-1 where p is the number of variables. If not given (=None) then alpha_omega is set to p+2.

  • prior_matrix (numpy.ndarray) – The prior matrix ‘T’ for the Wishart part of the normal-Wishart prior. If not given (=None), then this is set to t*I_n where t = alpha_mu*(alpha_omega-n-1)/(alpha_mu+1)

Type

float

bge_component(vs)¶

Compute the BGe component for given variables.

The BGe score for a family child<-parents is the component for child+parents minus the component for parents (+ a constant term which just depends on the number of parents).
