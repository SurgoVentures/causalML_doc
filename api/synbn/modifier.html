Add order systematic noise to pandas.DataFrame.

If the order of the potential answers is not randomised, then the respondent is more likely to choose the first options. For a specific variable, all the values after its third quartile are randomly changed with one of its level before the first quartile. :type data: DataFrame :param data: data :type data: pandas.DataFrame :type rates: Union[List[float], float] :param rates: replacement probability (between 0 and 1). To apply to each column/variable if noise_cols or num_noise_cols are specified :type rates: list or float :type noise_cols: Union[List[str], str, None] :param noise_cols: column name where to incude noise :type noise_cols: list or single string :type num_noise_cols: int :param num_noise_cols: number of columns where to incude noise (randomly defined) :type num_noise_cols: integer If ‘num_noise_cols’ and ‘noise_cols’ not specified, noise included in all variables of the dataset :rtype: DataFrame :return: data with systematic noise in column values :type: pandas.DataFrame
